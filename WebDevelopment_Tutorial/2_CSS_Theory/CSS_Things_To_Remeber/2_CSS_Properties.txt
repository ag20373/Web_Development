1. CSS background Properties :
    a. background :rgba(0, 128, 0, 0.3) /* Green background with 30% opacity */ //combination of background-color and opacity
    b. background-color : green; // Set background color to green
    c. opacity : 0.3; //set background color tranpacy
    d. background-image : url("image.jpg") //Sets the background image for an element
    e. background-repeat : repeat-x; // Set The repeatation property of image
    f. background-position : left top; //set the starting point of image
    g. background-attachment : scroll/fixed; //decides weather,we want our image scrollable or fixed on screen
    h. background-clip : value; //Specifies the painting area of the background
    i. background-origin : value; //Specifies where the background image(s) is/are positioned
    j. background-size : value; //Specifies the size of the background image(s)
    k. background : (color image repeat attachment position) //Short Cut For background.

6. CSS Height/Width Properties :
        1. auto - This is default. The browser calculates the height and width
        2. length - Defines the height/width in px, cm, etc.
        3. % - Defines the height/width in percent of the containing block
        4. initial - Sets the height/width to its default value
        5. inherit - The height/width will be inherited from its parent value
